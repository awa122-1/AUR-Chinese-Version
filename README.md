# 💡 安装方法：

### **如果你已经安装 BepInEx：**
下载 Releases 里的 `AUR.dll`，并放入：
`Among Us` → `BepInEx` → `Plugins`

### **如果你没有安装 BepInEx：**
- 下载 Releases 页面最新 `.zip`（暂未发布）
- 解压到 Among Us 游戏目录
- 直接覆盖文件

### **游戏目录位置：**
- **Steam：** 库 → 右键 Among Us → 管理 → 浏览本地文件
- **Epic：** 库 → 管理 → 文件夹图标
- **Itch.io：** 管理 → 打开文件夹
- **Xbox App：** 管理 → 文件 → 浏览
- **Microsoft Store：** 见[官方说明支持文章](https://answers.microsoft.com/en-us/xbox/forum/all/where-can-i-find-the-gamefiles-of-a-game/5cb9a0c3-7948-4316-abc5-f27d1767b932)

# 🎮 功能（v1.7.0）

## 🌟 自定义角色
- **市长**（船员）
- **小丑**（中立）

## ♠️ 客户端设置
- 游戏管理员
- 解锁 FPS / 显示 FPS
- 黑暗主题
- 大厅音乐开关
- 可自定义文本

## ⚙️ 技术设置
- 踢/封低等级玩家
- 踢/封无效 好友代码
- 踢说 “开或开始" 的玩家
- 禁词系统 / 黑名单
- 多层管理员系统
- `AUR-DATA` 添加管理员/高级会员（VIP）
- 不结束游戏模式
- 自定义开局倒计时

## ♻️ 自动化
- 自动开局
- 自动重进房间
- 自动发送胜利信息

## 🔧 游戏玩法
- 死亡内鬼可破坏
- 关闭各种破坏
- 禁止开局30秒开会
- 禁止报告尸体
- 地图功能禁用
- 自定义任务系统
- 捉迷藏模式优化

## 🏆 自定义模式
- 0冷却击杀
- 变形与躲猫猫
- 速通

## ❓ 其他优化
- 反转 Skeld 地图
- 可取消倒计时
- 更快启动
- 自定义反作弊
- 大厅/死亡缩放
- 死亡可看统计
- 高级选项范围
- **支持公开房间**

### ⌨️ 快捷键

| 快捷键 | 功能 | 场景 |
| :--- | :--- | :--- |
| `Shift` + `L` + `Enter` | 强制结束游戏 | 游戏中 |
| `Shift` + `M` + `Enter` | 跳过会议 | 游戏中 |
| `C` | 取消开局 | 倒计时 |
| `Shift` | 立即开局 / ×5 选项 | 倒计时 / 设置 |
| `Ctrl` | Noclip（穿墙） / ×10 选项 | 大厅 / 设置 |
